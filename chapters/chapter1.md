#include <iostream>

---
---
### What does each part mean?
---
```Markdown
    #include 
```
It is a C++ directive used to import libraries into your system.
---
---
```Markdown
    int main(){...}
```
Defining a function that must return a positive or negative integer in the context of creating the main function that libc needs to run your code correctly.
---
---
```Markdown
    std
```
It is the standard namespace for object-oriented usage in C++ of C++ I/O functions and other functions.
---
---
```Markdown
    return
```
Returns a value or something like pointers, functions, numbers, bytes, strings, arrays, and others in C/C++ and other similar languages.
